# 這是一個「魔法筆記本」系統

想像你有一個**超級聰明的機器人朋友**，他住在你的電腦裡。

你們的約定是這樣的：

> 你負責**想到什麼就說**，他負責**幫你整理好**。

---

## 機器人是電動牙刷，不是糖果

電動牙刷幫你把牙齒刷得更乾淨，但**你還是要自己刷**。用久了，你的牙齒變健康了——不是你變得更依賴牙刷。

糖果不一樣。吃了很爽，但吃太多你就離不開它，牙齒也壞掉了。

這個機器人朋友要當你的**電動牙刷**：

- 他學習**你的整理方式**，不是要你學他的方式
- 用久了，你會更了解自己喜歡怎麼記筆記
- 就算有一天不用他了，**你的筆記還是你的**

> 如果有一天你發現「沒有機器人就完全不會整理」，那就要停下來想一想了。

---

## 你的筆記本長什麼樣？

這個系統有幾個**抽屜**：

| 抽屜 | 像什麼 | 放什麼 |
|------|--------|--------|
| 📥 Inbox | 玩具箱 | 先把東西隨手丟進去 |
| 📋 Projects | 作業本 | 有截止日期的事，例如「下週要交的報告」 |
| 🌱 Areas | 每天要做的事 | 不會結束的責任，例如「保持房間整潔」 |
| 📚 Resources | 參考書架 | 可以反覆翻的知識，例如食譜、筆記 |
| 📦 Archives | 完成任務的獎盃櫃 | 做完的東西，留著紀念 |
| 🪞 Identity | 自我介紹卡 | 告訴機器人「你是誰、你的夢想是什麼」 |

---

## 機器人朋友會幫你做什麼？

**你只要說：「我想到一件事！」**，丟進玩具箱（Inbox）。

機器人會：

1. 讀一讀你丟進去的東西
2. 想一想：「這應該放哪個抽屜？」
3. 幫你整理好，放到對的地方

你不用記什麼規則。**讓機器人去想就好。**

---

## 機器人會越來越懂你

每次你們聊天，他會偷偷記住你的習慣：

- 你喜歡簡短的答案？→ 他記下來
- 你不喜歡他用表情符號？→ 他記下來
- 用一個月之後，他跟你說話的方式，就會跟你剛開始用時**不一樣**了

但有一條鐵律：**他不能偷偷改變「你是誰」的設定**，那個一定要你自己點頭。

---

## 你可以跟他說的咒語

| 咒語 | 發生什麼事 |
|------|-----------|
| `/journal` | 「幫我記錄今天做了什麼」 |
| `/intake` | 「幫我整理玩具箱！」 |
| `/reflect` | 「這一週我做了什麼？幫我整理成重點」 |
| `/maintain` | 「幫我檢查筆記本哪裡亂掉了」 |
| `/digest` | 「把你觀察到的我的習慣給我看，我來決定要不要認帳」 |
| `/writing` | 「用我的口氣，幫我寫一篇文章」 |

---

## 每天怎麼用？

就像寫日記一樣，三個步驟：

```
每天睡前
  /journal ← 說說今天做了什麼、學到什麼

有空的時候
  /intake  ← 讓機器人把筆記本整理好

每個星期
  /reflect ← 機器人幫你回顧這週，找出你進步的地方
```

---

## 最重要的一句話

> 你是老闆，機器人是助理。
>
> 他可以建議，但**重要的事情，一定要你拍板**。
