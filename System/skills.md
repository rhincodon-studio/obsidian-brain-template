# 技能系統

> 技能是預定義的複雜工作流
> 用一個命令觸發，AI 按預設流程執行
> 模型越強，技能執行質量越高

## 核心技能

### /intake

**觸發**：`/intake` 或「歸檔」

**作用**：
1. 掃描 `Inbox/` 下所有文件
2. 分析每篇內容的本質
3. 判斷 PARA 歸屬
4. 添加 frontmatter
5. 移動到目標目錄

---

### /maintain

**觸發**：`/maintain` 或「維護」

**作用**：
1. 全庫健康掃描
2. 找出過時標籤、斷裂引用、缺失說明文件
3. 檢測 frontmatter 與實際不一致
4. 輸出診斷報告
5. 提出修復方案

---

### /digest

**觸發**：`/digest` 或「整理」

**作用**：
1. 讀取 [[candidates]]
2. 展示 🔴🟡🟢 候選
3. 用戶確認後晉升到 [[Identity/PROFILE]] 或 [[OPERATING_RULES]]
4. 清理過期條目

---

### /writing

**觸發**：`/writing` 或「寫作」

**作用**：
1. 加載用戶的寫作風格規則
2. 加載相關方法論和案例庫
3. 按用戶風格執行寫作任務

---

### /reflect

**觸發**：`/reflect` 或「覆盤」

**作用**：
1. 聚合過去 7 天的 journal
2. 提煉關鍵洞察
3. 識別重複模式
4. 更新候選偏好池
