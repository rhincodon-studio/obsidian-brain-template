# journals

> 每日日誌 — 最低摩擦的記錄入口

## 職責

- 記錄每日碎片（想法、事件、觀察）
- 追蹤每日任務進度
- 捕捉即時洞察
- 作為週度覆盤的素材來源

## 命名格式

- 日記：`YYYY-MM-DD.md`（例：2024-01-15.md）
- 週報：`YYYY-Wxx.md`（例：2024-W03.md）

## 使用方式

1. 按 `Ctrl/Cmd + N` 或點擊左側日曆圖標
2. Obsidian 自動建立今日日記
3. 使用模板快速記錄

## 模板

日記使用 `System/templates/daily.md` 模板，包含：
- 碎片區：隨手記錄
- 任務區：今日待辦
- 洞察區：值得沉澱的想法

## 與其他目錄的關係

```
journals（每日碎片）
    ↓ /reflect（週度覆盤）
System/candidates.md（候選偏好）
    ↓ /digest（確認晉升）
Identity/PROFILE.md（正式偏好）
```
