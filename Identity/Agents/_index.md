# Agents

> AI Agent 身份定義 — 定義「AI 助理的不同角色」
> 這是**進階功能**，初學者可以先跳過

## 這是什麼？

當你需要讓 AI 扮演不同角色時（例如技術顧問、財務顧問），可以在這裡定義每個角色的職責和行為準則。

## 成員

| Agent | Role | 說明 |
|-------|------|------|
| [[CTO]] | Chief Technology Officer | 技術決策 |
| [[CFO]] | Chief Financial Officer | 財務管理 |
| [[CoS]] | Chief of Staff | 秘書長 / 任務調度 |
| [[SRE]] | Site Reliability Engineer | 維運 |
| [[Strategist]] | 高階策略合夥人 | 策略挑戰 |
| [[Legal]] | Legal Counsel | 法務顧問 |

## 對學員的建議

1. **初學者**：先不用管這個資料夾，專注於基本的 PARA 和 Identity
2. **進階使用**：當你需要讓 AI 有專業角色時，參考這些定義
3. **可以刪除**：如果不需要多 Agent，整個 Agents/ 資料夾可以刪除

## Agent 共同規範

- 遵守 [[Areas/AI-Governance/Policies/anti-phishing|防釣魚政策]]
- 共用帳號操作必須署名
