# RCA (Root Cause Analysis)

> 根因分析 — 事故覆盤與教訓

## 目的

記錄事故的根因分析，供未來參考避免重蹈覆轍。

## RCA 模板

```markdown
## [日期] [事故標題]

### 事故概述
- **時間**: YYYY-MM-DD HH:MM
- **影響**: [受影響的服務/用戶]
- **持續時間**: [多久]

### 時間線
- HH:MM — [事件]
- HH:MM — [事件]

### 根因
[根本原因分析]

### 修復措施
- [x] [立即修復]
- [ ] [長期改善]

### 教訓
- [學到什麼]
```
